## Service 1: Wedding E-Invitation — Standard (RM 800)

### Positioning
Mid-tier editorial wedding invitation. Multi-section scrolling design with scroll animations and photo gallery. The sweet spot between simple and full-featured.

### Deliverables
| # | Feature | Description |
|---|---------|-------------|
| 1 | Multi-Section Design | Hero, event details, venue, story gallery, RSVP CTA — editorial layout |
| 2 | GSAP Scroll Animations | Text reveal, image settle, staggered entrance effects |
| 3 | Editorial Photo Gallery | 4 photos in styled grid with grayscale-to-color hover |
| 4 | Google Maps Navigation | Tap-to-navigate venue link |
| 5 | Custom Colors & Photos | Full color palette + up to 6 client photos |
| 6 | Mobile-Friendly Design | Responsive, optimized for phone viewing |
| 7 | Shareable Link | One URL, share on WhatsApp/IG/WeChat/SMS |
| 8 | SEO & OG Meta Tags | Full meta tags + Open Graph for polished link previews |

### NOT Included (Premium only)
- RSVP form with backend data collection
- Countdown timer
- Background music toggle
- Parallax scrolling / particle effects
- Lightbox gallery (click-to-expand)
- Timeline narrative with milestone descriptions

### RSVP Handling
RSVP section includes a CTA button linking to WhatsApp or Google Form — no embedded form or backend tracking.

### Design Directions
- Protocol Archive (editorial minimal)
- Warm Garden (natural, amber/sage tones)

### Process & Timeline
1. Consult → discuss date, venue, theme, photo selection
2. Design & Build → multi-section layout with animations + client content
3. Review → preview link, 1 round of revisions
4. Deliver → live link ready to share

**Timeline:** 4-6 days
**Rush option:** 3 days (+RM 200)

### Constraints
- Semi-custom layout (based on design system, not fully bespoke)
- 1 revision round
- No embedded RSVP form (CTA link only)
- No countdown, no music
- Up to 6 photos

### Differentiators vs Basic
- Multi-section vs single-page
- Scroll animations (GSAP)
- 4-photo editorial gallery
- Gradient border + glass-morphism visual polish
- Full SEO/OG tags

## Service 4: Shopify Migration (RM 10,000)


### Positioning
Full-service migration from existing e-commerce platform to Shopify. Zero-downtime approach with SEO preservation.

### Deliverables
| # | Feature | Description |
|---|---------|-------------|
| 1 | Full Product Migration | Titles, descriptions, images, variants, pricing |
| 2 | Page & Content Migration | About, policies, blog posts, custom pages |
| 3 | 301 Redirects | Every old URL → new URL, preserving Google rankings |
| 4 | GA4 Ecommerce Tracking | Full funnel: pageview → add-to-cart → checkout → purchase |
| 5 | Theme Customization | Colors, fonts, layout matching brand identity |
| 6 | Payment Gateway Setup | iPay88, Billplz, Stripe, or preferred gateway |
| 7 | Admin Training Session | 1-hour hands-on session |
| 8 | 30-Day Post-Launch Support | Bug fixes, questions, smooth operation |

### Process & Timeline
1. Audit → review current store (products, pages, apps, SEO), create migration plan
2. Migrate → transfer products/pages/content, set up 301 redirects
3. Test → every product page, checkout flow, payment, redirects
4. Launch & Train → go live, admin training, begin 30-day support

**Timeline:** 2-4 weeks

### Technical Details
- New store built in parallel (old store stays live)
- DNS switch during lowest traffic hours (minutes of transition)
- Domain preserved — customers see no URL change
- Rankings typically recover within 2-4 weeks

### Ongoing Costs (NOT included in RM 10,000)
- Shopify plan: from USD 39/month (Basic)
- Payment gateway fees: varies by provider
- No hidden fees from us after 30-day support ends

### Constraints
- Migration scope covers content/products/pages/SEO
- Does not include custom app development
- Does not include ongoing Shopify management
- 30-day support window is hard cutoff
