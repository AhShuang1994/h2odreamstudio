# Wedding E-Invitation — Premium (RM 1,500)

## Positioning
Full wedding mini-website with interactive features. Multi-section scrolling experience with RSVP tracking, countdown, and background music.

## Deliverables
| # | Feature | Description |
|---|---------|-------------|
| 1 | Multi-Section Design | Hero, story timeline, event details, gallery, RSVP form |
| 2 | RSVP Form + Backend | Embedded form (name, guests, attendance, dietary) + Google Sheets/Supabase dashboard |
| 3 | Countdown Timer | Live JS countdown to wedding date in hero section |
| 4 | Background Music Toggle | Auto-play song with fixed mute/play button |
| 5 | Parallax & Particle Effects | Parallax scrolling + sparkle/particle canvas animations |
| 6 | Story Timeline | Milestone narrative with dates, descriptions, photos, connecting line |
| 7 | Photo Gallery + Lightbox | 6-8 photos in responsive grid with click-to-expand viewer |
| 8 | Custom Colors & Photos | Every element matches wedding palette |
| 9 | Shareable Link | One link for all channels + QR code ready |
| 10 | Full SEO & OG Tags | Complete meta, Open Graph, Twitter Card |

## Sections Breakdown
1. **Header** — couple initials, date, optional navigation
2. **Hero** — fullscreen photo bg, couple names, date, live countdown timer, particle overlay
3. **Our Love Story** — vertical timeline with 3+ milestones (photo + date + text), connecting line
4. **Event Details** — ceremony + dinner as glass-morphism cards (time, venue, dress code, Google Maps)
5. **Gallery** — 6-8 photos responsive grid + lightbox on click
6. **RSVP** — embedded form: name input, guests dropdown, accept/decline radio, dietary textarea, submit
7. **Footer** — music toggle, hashtag, "Made with Love"

## Design Directions
- Luxury Dark Theme (navy #0A1628 + gold #D4AF37)
- Garden Romance Theme
- Cinematic Editorial

## Process & Timeline
1. Consult → vision, theme, color mood, must-have features, story milestones
2. Design → custom multi-section layout with content + interactions
3. Review & Refine → live preview, 2 rounds of revisions
4. Launch → go live + sharing setup + RSVP tracking handover

**Timeline:** 5-7 days

## Constraints
- Custom layout (not just template swap)
- 2 revision rounds
- Sections are modular — client can add/remove
- No guest limit on RSVP
- Up to 10 photos

## Technical Requirements
- RSVP backend: Google Sheets API or Supabase
- Music: HTML5 Audio API with user interaction trigger
- Countdown: vanilla JS, target date configurable
- Particles: Canvas 2D or Three.js (depending on design)
- Lightbox: vanilla JS or lightweight library
