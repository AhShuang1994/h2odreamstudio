# Wedding E-Invitation — Standard (RM 800)

## Positioning
Mid-tier editorial wedding invitation. Multi-section scrolling design with scroll animations and photo gallery.

## Demo
`demos/wedding-basic-1.html` — Protocol Archive style, amber/stone palette

## Deliverables
| # | Feature | Description |
|---|---------|-------------|
| 1 | Multi-Section Design | Hero, event details, venue, story gallery, RSVP CTA |
| 2 | GSAP Scroll Animations | Text reveal, image settle, staggered entrance effects |
| 3 | Editorial Photo Gallery | 4 photos in styled grid with grayscale-to-color hover |
| 4 | Google Maps Navigation | Tap-to-navigate venue link |
| 5 | Custom Colors & Photos | Full color palette + up to 6 client photos |
| 6 | Mobile-Friendly Design | Responsive, optimized for phone viewing |
| 7 | Shareable Link | One URL, share on WhatsApp/IG/WeChat/SMS |
| 8 | SEO & OG Meta Tags | Full meta tags + Open Graph for polished link previews |

## NOT Included (Premium only)
- RSVP form with backend data collection
- Countdown timer
- Background music toggle
- Parallax scrolling / particle effects
- Lightbox gallery (click-to-expand)
- Timeline narrative with milestone descriptions

## RSVP Handling
CTA button linking to WhatsApp or Google Form — no embedded form or backend tracking.

## Design Directions
- Protocol Archive (editorial minimal)
- Warm Garden (natural, amber/sage tones)

## Sections Breakdown
1. **Header** — couple initials, date
2. **Hero** — fullscreen photo bg, couple names (GSAP word reveal)
3. **Event Details** — date, time, dress code
4. **Venue** — name, address, photo, Google Maps link
5. **Our Story** — 4-photo editorial grid (Fig. 01-04)
6. **RSVP** — heading, message, CTA button (external link)
7. **Footer** — couple names, date, "Made with Love"

## Visual Features
- Gradient border treatment on cards
- Glass-morphism card backgrounds
- Grayscale → color hover on photos
- GSAP ScrollTrigger text reveal animations
- GSAP image settle (opacity + scale) on scroll

## Process & Timeline
1. Consult → date, venue, theme, photo selection
2. Design & Build → multi-section layout with animations + client content
3. Review → preview link, 1 round of revisions
4. Deliver → live link ready to share

**Timeline:** 4-6 days
**Rush option:** 3 days (+RM 200)

## Constraints
- Semi-custom layout (based on design system, not fully bespoke)
- 1 revision round
- No embedded RSVP form (CTA link only)
- No countdown, no music
- Up to 6 photos
